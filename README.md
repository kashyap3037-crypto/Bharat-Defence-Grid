# 🇮🇳 Bharat Defence Grid

**Real-Time Strategic Defence Intelligence Platform**

---

## 🚀 Overview

**Bharat Defence Grid** is a modern web-based platform that provides a **structured, real-time view of India's defence ecosystem** — including Army, Navy, Air Force, and strategic missile systems.

Built with a focus on **clean UI, data organization, and military-inspired design**, this project showcases defence capabilities in a **professional dashboard format**.

---

## 🔥 Features

* 🛰️ **Live Defence News Feed**: Real-time intelligence stream focusing on Indian service sectors (Army, Navy, IAF, DRDO).
* ✈️ **Air Force Assets**: High-density categorization of Fighters, Transport, UAVs, and Helicopters.
* 🚢 **Naval Power**: Detailed matrix of Carriers, Destroyers, Submarines, and Corvettes.
* 🪖 **Army Strength**: Tactical view of Infantry Combat Systems, Tanks, Artillery, and Elite Special Forces.
* 🚀 **Missiles & Strategic Systems**: Tracking Ballistic, Cruise, and Air Defence systems.
* 🧭 **Tactical Navigation**: High-fluidity tab system for rapid cross-service data switching.
* 🌙 **Command Center UI**: Dark-themed, military-inspired dashboard with premium glassmorphism effects.
* 📱 **Tactical Fluidity**: Fully responsive design optimized for desktop and mobile command.

---

## 🧠 Tech Stack

* **Structure**: HTML5 (Semantic Architecture)
* **Styling**: Vanilla CSS3 (Custom Grid-based Matrix Layout)
* **Logic**: JavaScript (Real-time RSS Fetching & Dynamic UI State)
* **API**: Google News RSS via `rss2json` (10-minute refresh cycle)
